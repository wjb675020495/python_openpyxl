# Module os 

![image-20200418000744995](../res/os.PNG)
### 系统操作

```
os.sep     #系统路径的分隔符
os.name    #正在使用的工作平台
os.getenv  #环境变量名称
getcwd     #获取当前的路径
```



### 目录操作

```
os.listdir  #获取指定目录下的所有文件和目录名
os.mkdir    #创建一个目录文件
os.rmdir	#删除目录文件
os.chdir    #改变当前目录到指定目录
os.rename   #重命名目录名
```



### path

os.path.join

os.path.dirname
os.path.basename
